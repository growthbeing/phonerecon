# :iphone: PhoneRecon
Simple Python program to ```generate phone number details```

</div>


### Requirements
- Python 3.0
- pip

### Python Dependencies
- phonenumbers

### Install dependencies
```
pip install -r requirements.txt
```

### Run the program
```
python phonerecon.py
```
